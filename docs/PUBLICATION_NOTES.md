# Publication Notes

This public repository was curated from a local coursework folder after checking the actual files.

## Included

- MATLAB source files required to rerun the two selected cases.
- Input CSV files for Case 1.
- Selected result CSV/TXT files and figures that make the output inspectable without rerunning MATLAB.
- Concise run instructions and version notes.

## Excluded

- Course slides, problem statements and teacher-provided PDFs.
- Full submitted reports and materials containing student-id style submission packaging.
- Reference papers, borrowed/reference code folders, archives and local backup versions.
- `.omx` workflow state and local absolute paths.

The intention is to show MATLAB modeling, simulation and engineering problem-solving ability while keeping the public archive small and respectful of course materials.
